# Forml-Rails

Support for [Forml](https://github.com/texodus/forml) in Rails. 

## Installation

Add this line to your application's Gemfile:

    gem 'forml-rails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install forml-rails

## Usage

    $ rails generate forml MODEL
